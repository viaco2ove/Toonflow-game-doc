# 回溯功能测试

## TC-A08: 消息回溯标记验证

### 测试目的
验证安卓端能否正确识别可回溯的消息

### 测试步骤
1. 获取一系列消息
2. 检查每条消息的 canRevisit 字段
3. 验证最后一条消息的 canRevisit 为 false

### 预期结果
- ✅ 除最后一条外，所有消息都有 canRevisit: true
- ✅ 最后一条消息 canRevisit: false
- ✅ 消息列表中包含必要的 revisitData

### 安卓端数据模型
```kotlin
data class MessageItem(
  // ...
  val revisitData: JsonElement? = null,  // 回溯数据
)

// 检查是否可回溯
fun MessageItem.canRevisit(index: Int, total: Int): Boolean {
  return index < total - 1
}
```

## TC-A11: 回溯后继续游戏

### 测试目的
验证回溯后能否正常继续游戏

### 测试步骤
1. 执行回溯操作
2. 从回溯点继续调用 debugStep
3. 验证新消息正确添加

### 预期结果
- ✅ 可以正常调用 debugStep
- ✅ 新消息正确添加到消息列表
- ✅ 状态正确推进
- ✅ 历史消息被正确覆盖

### 安卓端状态管理
```kotlin
class GameViewModel : ViewModel() {
  private val messages = mutableListOf<MessageItem>()
  private var currentState: JsonElement? = null

  suspend fun revisitAndContinue(messageIndex: Int, playerContent: String) {
    // 1. 执行回溯
    repository.revisitMessage(sessionId, messageIndex)

    // 2. 重新获取消息列表
    messages.clear()
    messages.addAll(repository.getMessages(sessionId))

    // 3. 继续游戏
    val result = repository.debugStep(
      worldId = worldId,
      chapterId = chapterId,
      state = currentState!!,
      messages = messages,
      playerContent = playerContent
    )

    // 4. 更新状态
    currentState = result.state
    messages.addAll(result.messages)
  }
}
```

## TC-A12: 边界情况测试

### 测试目的
验证回溯功能的边界情况处理

### 测试步骤
1. 尝试回溯到最后一条消息（应失败）
2. 尝试回溯到不存在的消息索引（应失败）
3. 尝试在空消息列表时回溯（应失败）

### 预期结果
- ✅ 回溯到最后一条消息返回错误
- ✅ 无效索引返回错误
- ✅ 空列表返回错误
- ✅ 错误信息清晰明确

### 错误处理
```kotlin
try {
  repository.revisitMessage(sessionId, messageIndex)
} catch (e: Exception) {
  when {
    e.message?.contains("无法回溯") == true -> {
      // 显示错误提示
      showError("无法回溯到最后一条消息")
    }
    e.message?.contains("不存在") == true -> {
      // 显示错误提示
      showError("消息不存在")
    }
    else -> {
      showError("回溯失败: ${e.message}")
    }
  }
}
```

## 安卓端 UI 集成

### 回溯按钮显示
```kotlin
@Composable
fun MessageItem(
  message: MessageItem,
  index: Int,
  total: Int,
  onRevisit: (Int) -> Unit
) {
  Card {
    Column {
      Text(text = message.content)
      if (index < total - 1) {
        Button(onClick = { onRevisit(index) }) {
          Text("回溯到这里")
        }
      }
    }
  }
}
```
