# Toonflow 项目概览

> 最后更新：2026-04-15

## 一、项目定位

Toonflow 是一款 **AI 短剧/互动故事游戏一站式工具平台**，利用 AI 技术将小说自动转化为剧本，并结合 AI 生成的图片和视频，实现高效的短剧创作与多角色沉浸式 AI 故事游戏体验。

### 核心标语

- **AI 短剧工厂**：动动手指，小说秒变剧集！
- **AI 剧本 × AI 影像 × 极速生成**
- **多角色沉浸感 AI 故事游戏**

## 二、技术栈

| 层级 | 技术选型 |
|------|----------|
| 运行时 | Node.js 23+ / Electron |
| 语言 | TypeScript (strict 模式) |
| Web 框架 | Express 5 |
| 桌面端 | Electron 40 |
| 数据库 | SQLite3 (better-sqlite3) |
| ORM | Knex.js |
| AI SDK | Vercel AI SDK (ai) + 多厂商 Provider |
| 图像 | Sharp |
| 语音 | 阿里云 CosyVoice |
| 构建 | tsx + electron-builder |

### AI 模型支持

- **文本大模型**：OpenAI、Anthropic、Google、DeepSeek、智谱、Qwen、xAI 等
- **视频生成**：豆包、Sora 等
- **图像生成**：Nano Banana Pro 等
- **语音合成**：阿里云 CosyVoice

## 三、核心业务线

### 主线 1：AI 短剧制作流水线

```
小说文本 → 大纲提取 → 剧本生成 → 分镜制作 → 视频合成 → 成片输出
```

| 模块 | 说明 |
|------|------|
| 小说管理 | 导入、管理小说章节文本 |
| 大纲生成 | AI 从小说提取剧情大纲、故事线 |
| 剧本生成 | 按大纲生成结构化对白剧本 |
| 分镜制作 | 生成分镜提示词、画面设计 |
| 视频合成 | 调用 AI 视频 API 合成片段 |
| 素材管理 | 图像/视频素材库管理 |

### 主线 2：多角色 AI 故事游戏（近期重点）

```
世界观创建 → 章节设计 → 游戏会话 → 角色互动 → 叙事引擎驱动
```

这是项目目前最复杂的核心业务，拥有专属运行时引擎 (`game-runtime`)，包含：
- 叙事编排系统
- 章节进度状态机
- 条件触发器
- 任务进度跟踪
- 小游戏控制
- 记忆管理系统

## 六、前端架构

前端为独立的 Vue3 SPA 项目，位于 `Toonflow-game-web/` 仓库。

### 前端构建产物

构建后的文件放置在 `scripts/web/` 目录，由 Electron 加载显示。

### 开发模式

- 开发时直接修改前端源码，运行 `yarn dev` 查看效果
- 不要在开发时执行 `yarn build`，以免影响开发效率
- 发行版才使用构建后的 `scripts/web/index.html`

## 七、多环境配置

| 环境 | 配置文件 | 用途 | 特点 |
|------|----------|------|------|
| `dev` | env/.env.dev | 快速启动、快速测试 | 通用配置，上传 git |
| `local` | env/.env.local | 本机调试 | 私有配置，不上传 git |
| `prod` | env/.env.prod | 生产发布 | 走外部环境变量，上传 git |

### 端口说明

- 后端 API 服务默认端口：`60002`
- 前端开发服务器默认端口：根据前端项目配置

## 八、数据库概览

使用 SQLite3 作为本地数据库，核心表结构：

| 表名 | 用途 |
|------|------|
| t_user | 用户信息 |
| t_project | 项目信息 |
| t_novel | 小说章节 |
| t_outline | 剧情大纲 |
| t_script / t_scriptSegment | 剧本及片段 |
| t_assets | 素材库 |
| t_video / t_videoConfig | 视频及配置 |
| t_storyWorld / t_storyChapter | 故事世界/章节 |
| t_gameSession | 游戏会话 |
| t_sessionMessage | 会话消息 |
| t_sessionStateSnapshot | 会话状态快照 |
| t_entityStateDelta | 实体状态变更 |
| t_chapterTask / t_chapterTrigger | 章节任务/触发器 |
| t_config | AI 模型配置 |
| t_prompts | 提示词模板 |
